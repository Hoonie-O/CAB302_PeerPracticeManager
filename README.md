## Overview

The system allows users to:
- Create and manage **peer practice sessions**
- Register and authenticate user accounts
- View, join, and create group sessions
- Store and retrieve data from a **persistent SQLite database**

Developed by a team of five developers following **agile software engineering practices**.

## Getting Started

### Prerequisites
- **Amazon Corretto 21 (Java 21)**
- **JavaFX SDK**
- **Maven** (for building and dependency management)

### Running the Project
1. Clone this repository:
   ```bash
   git clone https://github.com/Hoonie-O/CAB302_PeerPracticeManager.git
   ```
2. Open the project in your preferred IDE (e.g. IntelliJ IDEA or Eclipse).
3. Ensure JavaFX and Maven dependencies are configured.
4. Build the project with:
   ```bash
   mvn clean install
   ```
5. Run the main application:
   ```bash
   mvn javafx:run
   ```
